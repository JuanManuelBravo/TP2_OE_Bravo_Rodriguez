Trabajo Practico 2 

Organizacion Empresarial

Integrantes: Juan Manuel Bravo
             Nicolas Rodriguez

Comision: 9

Se decidio realizar un programa para una pequeña empresa que analice las ventas y ayude a la evaluacion del desempeño de la empresa.
Para esto usamos el dataset de ventas simuladas, el cual contiene 3 columnas con ID, fecha de venta y monto.
Trabajando con esos datos el programa devolvera al usuario las ventas totales, las ventas por mas y graficos comparativos para analizar ventas mensuales.
