Comision: 9

Se decidio realizar un programa para una pequeña empresa que analice las ventas y ayude a la evaluacion del desempeño de la empresa.
Para esto usamos el dataset de ventas simuladas, el cual contiene 3 columnas con ID, fecha de venta y monto.
Trabajando con esos datos el programa devolvera al usuario las ventas totales, las ventas por mas y graficos comparativos para analizar ventas mensuales.

Instrucciones del script: 
Linux:
Activar el entorno virtual con el comando source .venv/bin/activate (linux)
Instalar las dependencias con pip install requirements.txt
Ingresar al directorio scripts y utilizar el comando python3 ./scripts/script_ventas.py  para ejecutar

Windows (PowerShell):
.venv\Scripts\Activate.ps1

Nota: Si PowerShell bloquea el script, ejecuta Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser.
